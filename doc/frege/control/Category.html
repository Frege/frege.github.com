Definitions

class Semigroupoid f ⇒ Category f

A category is a Semigroupoid with the following additional properties:

  • For every object A, there exists a mophism id(A): A -> A called the identity morphism for A, such that for every morphism f: A -> B we have
 id(B) • f = f = f • id(A)
